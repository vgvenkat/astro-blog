# Venkat's Tech Blog

A personal blog about web development, React, JavaScript, and other tech topics.

Built with [Astro](https://astro.build).

[![Netlify Status](https://api.netlify.com/api/v1/badges/46f7e710-8b9a-4ac9-b52d-fd82968b1792/deploy-status)](https://app.netlify.com/projects/venkatblog/deploys)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
